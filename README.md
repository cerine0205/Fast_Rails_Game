Fast Rails Game

Fast Rails is a fast-paced endless runner game inspired by subway-style mechanics.
The player runs through railway tracks, avoids trains and obstacles, and collects coins to achieve the highest possible score.
The game focuses on simple controls and smooth gameplay.

Project Objective

The goal of this project is to design a fun and responsive runner game that challenges player reflexes and timing, while demonstrating core game development concepts using Unity.

Features
• Endless runner gameplay
• Train and obstacle avoidance
• Coin collection system
• Simple and intuitive controls
• Score tracking

Gameplay
• The player runs automatically forward
• Move between tracks to avoid trains
• Collect coins to increase score
• The game ends when the player hits an obstacle

Technical Overview
• Developed using Unity
• Built with C#
• Collision detection and triggers
• Scene management and prefabs
• Optimized for smooth performance

Limitations
• Single game mode
• No character customization
• No save system for scores
